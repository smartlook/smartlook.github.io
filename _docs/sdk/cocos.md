## Project build

### Android

In case you want to use SDK in Android project add these lines to your gradle project file:

```android
allprojects {
  repositories {
    maven {
      url "https://sdk.smartlook.com/android/release"
    }
  }
}
```

Also add this line to your module gradle: `implementation('com.smartlook.recording:app:1.1.5-cocos')`
