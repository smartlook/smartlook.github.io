## Tracking code

Copy the tracking code from your project settings and include it on your web. The code should be similar to this:

<Code
  kind="example"
  snippets={{
    javascript: `
<script type='text/javascript'>
  window.smartlook||(function(d) {
    var o=smartlook=function(){ o.api.push(arguments)},h=d.getElementsByTagName('head')[0];
    var c=d.createElement('script');o.api=new Array();c.async=true;c.type='text/javascript';
    c.charset='utf-8';c.src='https://rec.smartlook.com/recorder.js';h.appendChild(c);
    })(document);
    smartlook('init', 'YOUR_PROJECT_KEY');
</script>
`,
  }}
/>

Check the `GDPR compliant by design` section and adjust the settings if needed. If you plan to identify visitors from EU you have to sign our DPA, read more on that in the GDPR section.
