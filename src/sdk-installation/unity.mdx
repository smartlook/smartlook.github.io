## Installation for Android - aar

1. Download [Smartlook Android Unity SDK v1.5.2](https://sdk.smartlook.com/android/app-1.5.2-unity.aar).
2. Put `AAR` file to `Assets/Plugins/Android` in your Unity project.
3. Get Unity bridge from [our Github repository](https://github.com/smartlook/unity-bridge).

## Installation for Android - Gradle

In case there is a conflict with other libraries, we also released lite version of the SDK. This solution however needs to be implemented via gradle of the app itself. Steps are described below. Flow is similar to integration of our native SDK.

1. Add Smartlook in your **appâ€™s** `build.gradle`:

<Code
  kind="example"
  snippets={{
    gradle: `
implementation('com.smartlook.recording:app:1.5.2-unity-lite')
`,
  }}
/>

2. Add Smartlook Maven repository in your **projects** `build.gradle`:

<Code
  kind="example"
  snippets={{
    gradle: `
allprojects {
  repositories {
    maven {
      url "https://sdk.smartlook.com/android/release"
    }
  }
}
`,
  }}
/>

3. In case you are missing some of the dependencies, feel free to add:

<Code
  kind="example"
  snippets={{
    gradle: `
implementation 'org.jetbrains.kotlin:kotlin-stdlib-jdk8:1.3.61'
`,
  }}
/>

4. Get Unity bridge from [our Github repository](https://github.com/smartlook/unity-bridge).

## Installation for iOS

1. Download [Smartlook iOS Unity SDK v1.3.1](https://sdk.smartlook.com/ios/smartlook-unity-ios-sdk-1.3.1.50.zip).
2. Unzip the archive and put `SmartlookUnity.framework` to `Assets/Plugins/iOS` in your Unity project.
3. Get Unity bridge from [our Github repository](https://github.com/smartlook/unity-bridge).

# Supported versions
Currently **minimal** supported Android version is **Jelly Bean (API 18)**, however **video is recorded only for API 24+**, analytics events and other functionality is not limited.

Smartlook Unity iOS SDK works with Metal graphics API only. OpenGL is not supported at the moment.

# Start Recording

To start recording (or just to setup the SDK), you can use i.e.  :

<Code
  kind="example"
  snippets={{
    csharp: `
public class NewBehaviourScript : MonoBehaviour
{
    // Start is called before the first frame update
    void Start()
    {
        SmartlookUnity.Smartlook.SetupAndStartRecording("API_KEY");
    }
\n
    // Update is called once per frame
    void Update()
    {
        ...
    }
}
`,
  }}
/>

<Alert kind="important">
To setup Smartlook, you need your unique API key. This can be found in your mobile project settings in <a href="https://app.smartlook.com/settings/projects">your Smartlook Dashboard</a>.
</Alert>

<TextBlock kind="important">
{`
SDK will upload sessions **only on WiFi**. When uploading on mobile connections is requested, too, it must be enabled in the project settings in the dashboard. More details about the recording process can be found in the <Link to="/conceptual/when-the-recordings-appear-in-dashboard">Conceptual documentation</Link>.
`}
</TextBlock>
