## Installation for iOS

1. Download Smartlook iOS Unreal Engine 4 SDK:
* [SDK for Unreal Engine 4.26 and newer](https://sdk.smartlook.com/ios/smartlook-unreal-ios-sdk-1.2.426.zip)
* [SDK for Unreal Engine 4.25 and older](https://sdk.smartlook.com/ios/smartlook-unreal-ios-sdk-1.1.zip)
2. Unzip the archive, create a `Plugins` directory in your project's directory (if needed), and put `Smartlook` to `Plugins`.
3. Open Plugins settings (Edit - Plugins) and enable Smartlook plugin.
<img visibleOn="unreal" src={unreal_enable} />

4. Set analytics provider to `IOSSmartlook` in Project Settings.
<img visibleOn="unreal" src={unreal_providers} />

6. Obtain your SDK Api Key from the [Smartlook Dashboard](https://app.smartlook.com) and add it into the Smartlook Plugin Settings. Optionally enable automatic recording by ticking the Start Recording option.
<img visibleOn="unreal" src={unreal_settings} />

# Supported versions

Currently the **minimal** supported Unreal Engine 4 version is **4.6.1** (Metal rendering only, openGL ES is not supported).
